# 📊 收益计算器 (Monthly Earn)

一个简约、美观的 30 天收益计算工具，帮助您快速估算理财产品的月度收益。

## ✨ 特性

- **实时计算**：输入本金或年化收益率时，收益结果即时更新。
- **美观 UI**：采用现代化的卡片式设计，具备毛玻璃效果和渐变背景。
- **响应式设计**：完美适配手机端和 PC 端。
- **精准计算**：按 30/365 比例计算 30 天收益，支持千分位格式化显示。

## 🚀 快速开始

1. 直接在浏览器中打开 `index.html`。
2. 输入您的**本金**（单位：万元）。
3. 输入**年化收益率**（单位：%）。
4. 即可在结果区域看到对应的 **30 天收益**。

## 🧮 计算公式

该工具使用以下公式进行计算：

```
30天收益 = 本金(万元) * 10000 * (年化收益率 / 100) * (30 / 365)
```

## 🛠️ 技术栈

- **HTML5**：页面结构
- **CSS3**：现代化样式，包含 Flexbox 布局、毛玻璃效果 (backdrop-filter) 和媒体查询。
- **JavaScript (Vanilla)**：原生 JS 实现实时计算逻辑，无外部依赖。

## 📝 备注

- 结果保留两位小数，并对整数部分进行千分位处理。
- 默认计算周期为 30 天。
